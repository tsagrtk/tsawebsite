Types of Cloud- SaaS

SaaS

SaaS stands for software as a service which refers to having an alternative way to access software, instead of more traditional methods.

How it works:

In the past, softwares would usually be purchased and then loaded onto a device, SaaS has a subscription where the software is hosted on the cloud and accessed on the internet. There are a lot of known softwares made from SaaS like: Office 365, google apps, salesforce, Citrix GoToMeeting, Cisco WebEx and Netflix.

Examples of where SaaS can be used for businesses:

  • Accounting and invoicing
  • Tracking sales
  • Planning
  • Performance monitoring
  • Communications

Benefits:

  • Process power is supplied by cloud
  • Applications can be used once the user subscribes
  • You pay for what you use
  • Additional storage is easy to access without having to install new hardware or software
  • Updates happen automatically and are free of change
  • Applications can be accessed by any internet device like desktops, smartphones and tablets
  • You can access the applications from any location
  • The applications can be changed to suit your needs
