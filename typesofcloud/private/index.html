Types of Cloud- Private

Public

With Private Computing, only one customer/organization uses The Cloud.

How it works:

The cloud is externally hosted, but it is not shared with others, which is what makes it private. Because it is private, the security and control level is highest. The cost can be very low if the company invests in an on-premise cloud infrastructure, but if the company buys it but does not invest into it, the costs soar.

Benefits:

  • Most Security
  • Moderately speedy
  • Elastic resource capability
  • Virtualized resources
  • Management automation
  • Self-service provisioning
